## 📖 About this application

Codex-bridge serves as a central hub for your AI tools. It creates a bridge between your computer and various artificial intelligence services. Instead of opening multiple browser tabs to talk to different AI systems, you use this single desktop manager. You can access models like GPT, DeepSeek, and Kimi within one interface.

The application simplifies how you switch between models. You provide your API keys to the system, and the interface handles the communication. This keeps your workflow efficient and reduces the need to track different website logins for each AI provider.

## ⚙️ Initial setup

When you open the application for the first time, you must configure your settings. The software needs access keys for the services you intend to use.

1.  Open the Settings menu inside the app.
2.  Select the Provider tab.
3.  Choose the service you want to add, such as GPT or DeepSeek.
4.  Paste your personal API key into the input field. You get these keys from the account dashboard of the respective AI service website.
5.  Save your settings. The application verifies your keys and displays the available models in your dashboard.

## ⚡ Using the interface

The main window shows your active chat area. At the top of the interface, you see a dropdown menu. Use this menu to select which AI model you want to use for your current task. 

When you type a message and press Enter, the software sends your input to the selected model. The response appears in the chat window. You can change models at any time by selecting a different option from the dropdown menu. This allows you to compare answers from different AI systems without leaving the app.

## 🛡️ Security and privacy

Your API keys stay on your computer. The application stores them in a local file that is not accessible to other users or the software developer. All traffic between your computer and the AI providers uses encryption for your safety. Review your API key usage regularly on the websites of your chosen providers to ensure your accounts remain secure.

## 🛠️ Troubleshooting common issues

If the application fails to connect, check your internet settings first. Sometimes, firewall software prevents the application from reaching the internet. If you see an authentication error, double-check your API key in the Settings menu.

Ensure you have the latest version of the application. Developers release updates to add models and fix errors. If you encounter a persistent issue, restart the application. This often resolves minor connection glitches.
